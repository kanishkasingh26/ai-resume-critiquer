# AI Resume Critiquer

An AI-powered resume analysis tool built using Python, Streamlit, and Google's Gemini API.

## Features
- Upload PDF resumes
- Extract resume text automatically
- Get AI-generated feedback
- Receive role-specific suggestions

## Tech Stack
- Python
- Streamlit
- Gemini API
- PyPDF2
- python-dotenv
